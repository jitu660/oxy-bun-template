# 🚀 Quick Start

> **⚠️ Important:** Run the upgrade command immediately after cloning to ensure you are using the latest dependencies.

## 1. Setup
```bash
bun install
bun run upgrade
```
## 2. Quality Control
- Format: `bun run fmt` (Uses oxfmt)
- Lint: `bun run lint` (Uses oxlint)
- Analyze: `bun run unused` (Uses knip)
